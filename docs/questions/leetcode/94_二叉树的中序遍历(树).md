树的遍历, 题目要求用迭代法.
```
# 递归
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res = []
        self.traverse(root, res)
        return res

    def traverse(self, root, res):
        if root == None:
            return
        self.traverse(root.left, res)
        res.append(root.val)
        self.traverse(root.right, res)
```

迭代法就是用栈来做一个dfs.
```
# 迭代
class Solution:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res = []
        stack = []
        while stack or root != None:
            while root != None:
                stack.append(root)
                root = root.left
            root = stack.pop()
            res.append(root.val)
            root = root.right
        return res
```
