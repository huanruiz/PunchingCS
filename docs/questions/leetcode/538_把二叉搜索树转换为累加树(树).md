二叉搜索树自然就想到中序遍历, 累加其实就是反过来的中序遍历, 先搜右节点, 再搜左节点.
```
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def convertBST(self, root: TreeNode) -> TreeNode:
        s = 0
        if root:
            self.convertBST(root.right)
            s += root.val
            root.val = s
            self.convertBST(root.left)
        return root
```
