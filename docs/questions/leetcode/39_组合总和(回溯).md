组合题, 用回溯. 这里可以对candidates排序, 如果target减去现在的candidates已经小于0了, 那么更大的candidates也不可能可以用了, 故剪枝.
```
class Solution {
    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        List<List<Integer>> res = new ArrayList<>();
        ArrayDeque<Integer> path = new ArrayDeque<>();

        Arrays.sort(candidates);
        dfs(res, path, 0, target, candidates);
        return res;

    }

    public void dfs(List<List<Integer>> res, ArrayDeque<Integer> path, int left, int target, int[] candidates) {
        if (target == 0) {
            res.add(new ArrayList(path));
        } else if (target < 0) {
            return;
        }

        for (int i = left; i < candidates.length; i++) {
            if (target - candidates[i] < 0) break;
            path.addLast(candidates[i]);
            dfs(res, path, i, target - candidates[i], candidates);
            path.removeLast();
        }
    }
}
```
