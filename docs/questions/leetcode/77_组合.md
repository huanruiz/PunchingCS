组合问题用回溯解决, 分两种方法, 第一种是dfs直接遍历, 第二种是每次判断选或不选这个数, 我用第一种写的. 能想办法剪枝就剪枝.
```
class Solution {
    public List<List<Integer>> combine(int n, int k) {
        List<List<Integer>> res = new ArrayList();
        ArrayDeque<Integer> path = new ArrayDeque<>();

        dfs(1, n, k, res, path);
        return res;
    }

    public void dfs(int begin, int n, int k, List<List<Integer>> res, ArrayDeque<Integer> path) {
        if (path.size() == k) {
            res.add(new ArrayList(path));
            return;
        }

        for (int i = begin; i <= n - (k - path.size()) + 1; i++) {
            path.add(i);
            dfs(i + 1, n, k, res, path);
            path.removeLast();
        }
    }
}
```
